# Power Fitness Club

A modern fitness club business website developed using HTML, CSS, and JavaScript.

## Features

- Responsive Design
- Modern Dark Slate + Orange UI
- Hero Section
- About Section
- Fitness Programs
- Trainer Profiles
- Animated Statistics Counters
- Testimonials Slider
- Membership Plans
- Contact Form
- Google Maps Integration
- Mobile Navigation
- Scroll Reveal Animations

## Technologies Used

- HTML5
- CSS3
- JavaScript
